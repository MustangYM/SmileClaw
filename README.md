# SmileClaw
Rewrite a mini OpenClaw LLM Agent.
## USAGE
Create a.env file in the root directory and then add your own key
```
DEEPSEEK_API_KEY=skxxxxxxxxxxx
or
OPENAI_API_KEY=sk-proj-xxxxxxxx
```
